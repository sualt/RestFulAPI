âš™ï¸ Getting Started
1. Prerequisites
Make sure you have:

.NET SDK installed

Node.js (LTS recommended) installed

A running SQL Server instance (localdb or full SQL Server)

2. Backend Setup (ASP.NET Core Web API)
From the repository root:

dotnet restore
dotnet ef database update    # optional, if you want to apply migrations
dotnet run
By default, the API typically runs on:

https://localhost:xxxx

http://localhost:xxxx

(You can check/adjust the exact ports in launchSettings.json.)

Swagger UI is available at:

https://localhost:xxxx/swagger
3. Frontend Setup (React Vite Client)
Open a new terminal and navigate to the React client folder:

cd react-rest-client
npm install
npm run dev
By default, Vite serves the frontend at:


http://localhost:5173
If needed, update the API base URL in your React code (e.g. in src/App.jsx or a config file):


const API_BASE_URL = "https://localhost:xxxx"; // match your backend port
